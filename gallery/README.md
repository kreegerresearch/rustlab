# rustlab Notebook Gallery

Twelve worked notebooks demonstrating rustlab's plotting, DSP, numerical, and
notebook-system features. Each one is rendered Markdown — what you see here
is the actual executed output of the source notebook, with text results and
SVG plots inline. GitHub renders these directly.

The source notebooks live in [`examples/notebooks/`](../examples/notebooks/);
this directory is regenerated by `make notebooks` from the repo root.

## Plotting and visualization

| Notebook | What it shows |
|----------|----------------|
| [Contour Plots](contour_plots.md) | `contour` and `contourf` for level curves of 2-D scalar fields — equipotentials, isobars, density plots |
| [Surface Plots](surface_plots.md) | 3-D surfaces with `meshgrid` and `surf`, colormap selection, view angles |
| [Quiver and Streamplot](vector_fields.md) | Discrete arrow fields and continuous streamlines for 2-D vector fields |
| [String Arrays and Categorical Charts](string_arrays.md) | Categorical bar charts using string arrays as x-axis labels |

## Digital signal processing

| Notebook | What it shows |
|----------|----------------|
| [Lowpass Filter Analysis](filter_analysis.md) | 64-tap FIR lowpass design, frequency response, group delay |
| [FIR Filter Quantization: firpm vs firpmq](firpm_quantization.md) | `firpm` floating-point design vs naïve quantization vs `firpmq` integer-domain optimization at 10-bit precision |
| [Spectral Estimation](spectral_estimation.md) | Periodogram vs windowed PSD, Hann vs Hamming, variance reduction via averaging |

## Numerical methods

| Notebook | What it shows |
|----------|----------------|
| [The Sparse Laplacian](laplacian.md) | The 5-point Laplacian stencil as a sparse matrix; Poisson solve via `spsolve` |
| [Vector Calculus on Uniform Grids](vector_calculus.md) | Numerical `gradient`, `divergence`, `curl` on 2-D and 3-D grids |

## Notebook system

| Notebook | What it shows |
|----------|----------------|
| [Quick Look: Random Data](quick_look.md) | Minimal one-block notebook — the smallest useful example |
| [Template Interpolation](template_interpolation.md) | `${expr}` syntax to embed computed values in prose; sprintf-style format specs |
| [Multi-Notebook Rendering](multi_notebook.md) | Directory builds with cross-notebook links and the generated index page |

## Local interactive view

For Plotly hover/zoom/pan and KaTeX-rendered math, run `make notebooks` then
open `gallery/index.html` (or any `gallery/<name>.html`) in a browser. The
`.html` files are gitignored — regenerate them locally any time.
