# rustlab Notebook Gallery

Worked notebooks demonstrating rustlab features. Each one is rendered
Markdown — what you see here is the actual executed output of the source
notebook, with text results and SVG plots inline. GitHub renders these
directly.

The source notebooks live in [`examples/notebooks/`](../examples/notebooks/);
this directory is regenerated by `make notebooks` from the repo root.

Notebooks are grouped by toolbox. The same 12-toolbox taxonomy is used
throughout the project: in the REPL (`help`, `help <toolbox>`), the CLI
(`rustlab docs <toolbox>`), and the `examples/<toolbox>/` script tree.

## Local interactive view

For Plotly hover/zoom/pan and KaTeX-rendered math, run `make notebooks` then
open `gallery/index.html` (or any `gallery/<name>.html`) in a browser. The
`.html` files are gitignored — regenerate them locally any time.
